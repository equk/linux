## Main Apps

- Web Browser: Firefox
- File Manager: Thunar
- Terminal: alacritty + tmux
- Twitter Client: Tweetdeck Web
- Desktop Stats: Conky + i3bar
- Desktop Background: Nitrogen
- Text & Code Editor: vim & vscode
- Image Editor: Gimp
- Video Playback: VLC
- Launcher: dmenu
- Music Playback: Deadbeef & Spotify Web
- Email Client: Thunderbird + Enigmail
- IRC: Hexchat / irssi
